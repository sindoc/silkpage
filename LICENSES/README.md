# License Partition

SilkPage does not have a single uniform license across every file in the
repository. This directory defines the intended split by artifact class.

- `SOFTWARE.md`: core software and build logic
- `DOCUMENTATION.md`: manuals and technical documentation
- `CONTENT.md`: website copy, marketing pages, and editorial content
- `TEMPLATES.md`: starter sites, templates, and example assets
- `TRADEMARKS.md`: trademark and branding rules

Where a file or subtree is explicitly marked otherwise, that specific notice
controls.

This partition describes project intent and repository organization. It should
be aligned with actual release packaging and legal review before a commercial
program is announced as final.
