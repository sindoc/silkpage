# Templates License Scope

Intended scope:

- `templates/`
- starter websites
- example themes and sample site assets

Current repository intent:

- templates may remain separately licensed from the core software
- if a template is intended to be dual-licensed with the software, that should
  be stated explicitly in the template package or release metadata

Notes:

- Template images, CSS, and editorial copy may involve different rights from
  executable software logic.
