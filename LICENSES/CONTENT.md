# Website Content License Scope

Intended scope:

- project website copy
- editorial pages
- announcements
- branding and marketing content

Current repository intent:

- content licensing may remain separate from software licensing
- historical SilkPage site content references Creative Commons terms in several
  places and should be reviewed before any commercial redistribution package is
  finalized

Notes:

- This category should not be assumed to fall into the commercial software
  license unless explicitly stated.
