About

This site is about a powerfull web publishing framework called SilkPage. Web publishing tools are certainly plentiful and have been available for a long time but sadly none of them met our requirements and design goals.

Mission

We came together to develop an outstanding Web Publishing Framework built around XML markup languages and Web Standards. Do you want to participate?
