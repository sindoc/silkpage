Developers

SilkPage code repository and issue tracker are hosted at java.net. To become a developer, please request a developer role. As a developer, you can:

  • contribute to the SilkPage code repository;

  • report a bug;

  • post to SilkPage mailing-lists.

How to checkout SilkPage from java.net

  1. cvs -d:pserver:guest@cvs.dev.java.net:/cvs login

  2. cvs -d:pserver:guest@cvs.dev.java.net:/cvs co silkpage/[modulename]
