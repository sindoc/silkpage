What's SilkPage?

SilkPage is an XML based Web publishing framework that has evolved from the Docbook Website. With strong foundations in XML, it is a publishing framework with a focus on web standards and usability. SilkPage provides a framework for publishing structured and standard compliant websites in XML. It is particularly suited to publish personal websites, though it is by no means limited to them.
